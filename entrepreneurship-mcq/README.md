# Entrepreneurship MCQ Trainer

Open `index.html` to learn + memorize MCQs chapter-wise.

## What you get

- **Learn mode**: flashcard-style, with spaced repetition “boxes” (your progress is saved).
- **Test mode**: unit-wise tests with instant feedback + final score.
- **Due mode**: quickly study questions that are “due” for revision.
- **Notes + favourites**: save short memory notes per MCQ.
- **Light/Dark UI**: toggle from the top bar (saved).

## How to run

Just open:

- `entrepreneurship-mcq/index.html`

No installation needed.

## Data

Questions are parsed from the pasted text inside:

- `entrepreneurship-mcq/questions.js`

If you want **exactly 250 MCQs**, paste the remaining ones (your message currently parses fewer), and I’ll append them so the app loads all 250.
